## Firebase Contact Form

### Add your firebase project credentials in main.js file

 ```sh
var config = {
  apiKey: "",
  authDomain: "",
  databaseURL: "",
  projectId: "",
  storageBucket: "",
  messagingSenderId: "",
  appId: "",
};
firebase.initializeApp(config);
   ```
